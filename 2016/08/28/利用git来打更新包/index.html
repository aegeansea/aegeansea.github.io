利用git来打更新包

  • git
  • 打包

OLD_COMMIT_ID_HERE仅作为打包的起始点，但并不包含OLD_COMMIT_ID_HERE提交的内容。

当然NEW_COMMIT_ID_HERE 和OLD_COMMIT_ID_HERE之间可以间隔多个COMMIT的，这样就会打多个COMMIT的内容打包到一个压缩包内。
